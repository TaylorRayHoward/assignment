# Assignment

This project includes:

- `main.rb`: package sorting logic
- `tests.rb`: RSpec tests

## Prerequisites

- macOS, Linux, or Windows
- Ruby installed
- `gem` available

## 1) Verify Ruby is installed

```sh
ruby -v
```

If that command fails, install Ruby first

## 2) Install RSpec

```sh
gem install rspec
```

## 3) Run the tests

From the project directory:

```sh
ruby tests.rb
```
